# Planner program(플래너 프로그램)

심플하고 강력한 웹 기반 일정 관리 애플리케이션입니다.  
아이폰 스타일의 시간 설정, 드래그 인터랙션, 중요도 표시 등 사용자 편의성을 고려한 기능들을 Vanilla JS로 구현했습니다.

## ✨ 주요 기능 (Features)

### 1. 캘린더 뷰 (Monthly Calendar)
- **월별 이동**: 이전/다음 달로 부드럽게 이동하며 일정을 확인할 수 있습니다.
- **오늘 이동**: 언제든 'Today' 버튼으로 현재 달로 복귀합니다.
- **공휴일/주말 표시**: 대한민국 공휴일(설날, 추석 등)과 주말(토/일) 색상 구분.

### 2. 일정 관리 (Event Management)
- **일정 추가**: 날짜를 클릭하여 모달 창에서 일정을 등록합니다.
- **카테고리/중요도**:
    - **급함 (Urgent)**: 빨간색 느낌표(!) 아이콘 표시
    - **중요 (Important)**: 노란색 별(★) 아이콘 표시
- **완료 처리**: 일정 목록에서 체크박스 스타일의 완료(취소선) 처리 가능.
- **자동 저장**: 모든 데이터는 브라우저(`localStorage`)에 자동 저장되어 새로고침 해도 유지됩니다.

### 3. iOS 스타일 시간 선택기 (Time Picker)
- 기본 드롭다운(`select`) 대신 **스크롤 방식의 시간 선택 UI**를 자체 구현했습니다.
- **Scroll Snap**: 부드럽게 눈금에 걸리는 스크롤 경험.
- **Drag & Drop**: 마우스로 끌어서 시간을 빠르게 설정할 수 있는 제스처 지원.

### 4. 시각적 타임라인 (Visual Timeline)
- 하루의 일정을 시간 순서대로 막대 그래프 형태로 시각화하여 보여줍니다.
- 일정 길이에 따라 막대 높이가 자동으로 조절됩니다. (30분 단위)

## 🛠 기술 스택 (Tech Stack)
- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Icons**: FontAwesome 6
- **Fonts**: Google Fonts (Inter)
- **API**: (Optional) 기상청 중기예보 API 연동 코드 포함

## 🚀 실행 방법 (How to Run)
1. 프로젝트 폴더를 내려받습니다.
2. `index.html` 파일을 웹 브라우저(Chrome, Safari 등)에서 엽니다.
3. 별도의 서버 설치 없이 즉시 사용 가능합니다.
